# 네이버 검색 API 발급 가이드

이 프로젝트는 네이버의 **검색(Search) API**를 사용합니다. (지역 맛집 검색용)

## 1. 네이버 개발자 센터 접속
1. [네이버 개발자 센터](https://developers.naver.com/main/)에 접속합니다.
2. 네이버 아이디로 로그인합니다.

## 2. 애플리케이션 등록 (API 키 발급)
1. 상단 메뉴에서 **Application > 애플리케이션 등록**을 클릭합니다.
2. **애플리케이션 이름**: (예: `점심맛집파인더`) 입력
3. **사용 API**:
   - **검색** (Search) 을 선택합니다.
   - (선택) **네이버 로그인** 등 다른 건 불필요합니다.
4. **비로그인 오픈 API 서비스 환경**:
   - **WEB 설정**을 선택합니다.
   - **웹 서비스 URL(최대 10개)**에 `http://localhost:8501` 을 입력합니다. (Streamlit 기본 포트)
   - 등록하기 버튼을 누릅니다.

## 3. Client ID 및 Client Secret 확인
1. 등록이 완료되면 **Application > 내 애플리케이션** 메뉴로 이동합니다.
2. 방금 등록한 애플리케이션을 선택합니다.
3. 개요 탭에서 **Client ID**와 **Client Secret**을 확인합니다.
   - Client Secret은 "보기" 버튼을 눌러야 보입니다.

## 4. 프로젝트에 적용
1. 프로젝트 루트의 `.env` 파일을 엽니다. (없다면 `.env.example`을 복사해서 `.env`로 만드세요)
2. 아래와 같이 붙여넣습니다.

```bash
NAVER_CLIENT_ID=여러분의_Client_ID
NAVER_CLIENT_SECRET=여러분의_Client_Secret
```

3. 저장 후 앱을 재시작하면 데모 데이터 대신 **실제 데이터**가 조회됩니다!

> **주의**: 하루 허용량(Daily Limit)은 25,000회입니다. 개인 테스트용으로는 충분합니다.
